# CodeAlpha_CyberSecurity_Internship

This repository contains my completed tasks for the CodeAlpha Cybersecurity Internship. These projects demonstrate my ability to apply security principles to networking and software development, with a specific focus on systems relevant to Electrical Engineering and Embedded Linux.

---

## 📁 Repository Structure

* **/Task_1_Network_Sniffer:** A Python-based tool using the Scapy library to capture and analyze live network traffic (IP addresses and Protocols).
* **/Task_2_Phishing_Awareness:** A professional educational resource covering HTTPS myths, Homograph attacks, and Social Engineering prevention.
* **/Task_3_Secure_Coding_Review:** A security audit of Python code, identifying vulnerabilities like hardcoded secrets and providing secure remediations.

---

## 🛠️ Skills Demonstrated
* **Networking:** Packet inspection, TCP/IP protocols, and traffic monitoring.
* **Security Auditing:** Manual code review and risk assessment.
* **Programming:** Python scripting for automation and security.
* **Environment:** Linux (Ubuntu) terminal and VS Code.
